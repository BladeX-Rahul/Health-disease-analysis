# 🏥 Health-disease-analysis
#📌 Project Overview
This project presents a comprehensive Healthcare Data Analysis Dashboard built using Power BI. The objective is to analyze patient health data and identify key risk factors contributing to major diseases such as heart disease, diabetes, cancer, and tuberculosis.
The dashboard provides actionable insights into how lifestyle habits and demographic factors influence overall health conditions.
# 📊 Dataset Description
The dataset contains patient-level health records with the following attributes:---Demographics: Age, Gender
Health Metrics: BMI, Blood Pressure, Cholesterol
Lifestyle Factors: Smoking, Alcohol Consumption, Exercise
Medical Conditions: Heart Disease, Diabetes, Stroke, Cancer, Tuberculosis, Kidney Disease
Family History: Presence of hereditary diseases
# 🎯 Project Objectives
Analyze the distribution of diseases across different age groups and genders
Identify the impact of lifestyle factors like smoking and exercise on health
Evaluate key health indicators such as BMI, blood pressure, and cholesterol
Provide insights for early detection and risk assessment
