# 🏥 Health-disease-analysis
# 📌 Project Overview
This project presents a comprehensive Healthcare Data Analysis Dashboard built using Power BI. The objective is to analyze patient health data and identify key risk factors contributing to major diseases such as heart disease, diabetes, cancer, and tuberculosis.
The dashboard provides actionable insights into how lifestyle habits and demographic factors influence overall health conditions.
# 📊 Dataset Description
The dataset contains patient-level health records with the following attributes:---Demographics: Age, Gender
Health Metrics: BMI, Blood Pressure, Cholesterol
Lifestyle Factors: Smoking, Alcohol Consumption, Exercise
Medical Conditions: Heart Disease, Diabetes, Stroke, Cancer, Tuberculosis, Kidney Disease
Family History: Presence of hereditary diseases
# 🎯 Project Objectives
Analyze the distribution of diseases across different age groups and genders
Identify the impact of lifestyle factors like smoking and exercise on health
Evaluate key health indicators such as BMI, blood pressure, and cholesterol
Provide insights for early detection and risk assessment
# 📈 Key KPIs---
Average BMI
High Blood Pressure Count
High Cholesterol Count
Gender Distribution
# 📊 Dashboard Features---
Disease Analysis by Age Group & Gender
Impact of Smoking on Cancer & Tuberculosis
Blood Pressure Distribution by Exercise
Cholesterol & Parkinson’s Analysis by Family History
Interactive visualizations including bar charts, donut charts, and KPI cards
# 🔍 Key Insights---
Senior citizens show higher prevalence of chronic diseases
Smoking is strongly associated with cancer and tuberculosis cases
High BMI and high blood pressure indicate increased health risks
Lifestyle factors significantly influence disease occurrence
# 🛠️ Tools & Technologies Used---
Power BI – Data visualization & dashboard creation
DAX (Data Analysis Expressions) – Measures and calculations
Excel / CSV – Data source
# 🚀 How to Use---
Open the Power BI .pbix file
Explore the dashboard using filters and slicers
Analyze trends across different demographics and health indicators
# 📌 Conclusion

This project demonstrates how data analytics can be used in healthcare to uncover patterns, assess risks, and support better decision-making. It highlights the importance of preventive healthcare through data-driven insights.
